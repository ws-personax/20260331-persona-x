# LUCIA_CLOSE 사양

## 목적

`LUCIA_CLOSE`는 감정 대화 이후, ECHO의 정리 다음에 LUCIA가 짧고 따뜻하게 건네는 마무리 메시지입니다. 사용자가 바로 다음 입력으로 넘어가기 전에 대화의 온도를 부드럽게 낮추는 역할만 합니다.

## 트리거 조건

- `emotional`: 항상 실행
- `complex`: 감정 주제가 포함된 경우에만 실행
- `invest`: 실행하지 않음
- `casual`: 실행하지 않음

## 출력 형식

- 1줄: LUCIA의 따뜻한 반응 1~2문장
- 2줄: 부드러운 질문 1문장
- 전체 50단어 이내
- 태그: `[LUCIA_CLOSE]`

## 톤 가이드

- 분석 금지
- 숫자 사용 금지
- `마음`이라는 단어 직접 사용 금지
- 따뜻한 계절 은유는 사용 가능
- 조언보다 곁에 머무는 느낌 우선
- 결론을 강하게 밀지 않고, 사용자가 다음 말을 고를 여지를 남김

## 표시 위치

- ECHO 응답 이후 표시
- 사용자 다음 입력창 이전에 표시
- ChatWindow에서는 별도의 LUCIA 말풍선으로 표시

## 예시 응답

```text
[LUCIA_CLOSE]
오늘은 찬 바람 속에서도 작은 불빛 하나를 지켜낸 날 같아요.
지금 가장 먼저 내려놓고 싶은 말은 무엇인가요?
```

```text
[LUCIA_CLOSE]
오래 붙잡고 있던 생각이 잠시 의자에 앉은 밤처럼 느껴져요.
그중 하나만 천천히 다시 바라볼까요?
```

```text
[LUCIA_CLOSE]
봄비가 먼지를 가라앉히듯, 지금의 침묵도 제 역할을 하고 있을지 몰라요.
조금 더 말하고 싶은 장면이 남아 있나요?
```

## 구현 메모

- 오케스트레이션 결과에 `[LUCIA_CLOSE]` 태그를 포함할 수 있음
- API 응답에는 선택 필드로 분리 가능
- ChatWindow에서는 기존 ECHO 말풍선 뒤에 별도 LUCIA 말풍선으로 렌더링
- 기존 페르소나 프롬프트, 호출 구조, 환경 변수는 수정하지 않음
