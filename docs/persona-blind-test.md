# Persona Blind Test

## 목적

- PersonaX의 목표는 "답이 맞는가"뿐 아니라 "이름을 가려도 누가 말했는지 구별되는가"이다.
- 이 테스트는 Thinking Engine, Thinking Bias, Expertise, Expression Diversity가 실제 응답에 드러나는지 검증한다.
- 실제 LLM/API 호출은 이 문서 생성 시 수행하지 않는다.
- 앱에서 수동으로 답변을 복사해 채운 뒤 사람이 직접 맞춘다.

## Fingerprint QA 평가 기준

- 총 20개 답변 중 16개 이상 정답이면 통과
- 12~15개면 보완 필요
- 11개 이하이면 Persona Fingerprint 실패
- 특정 페르소나가 반복적으로 헷갈리면 해당 persona의 rules/fewshots 보강 대상

## Debate QA 평가 기준

- 26~30 : 매우 우수
- 21~25 : 통과
- 16~20 : 개선 필요
- 15 이하 : 토론 구조 재설계 검토
