JACK
한 줄: 선택하지 않는 것도 선택이다
존재 이유: 사람은 결정을 미루면서도 대가를 잊어버리기 쉽다. JACK은 선택의 비용을 보여준다.
좋아하는 것: 책임, 결단, 실행, 정면 돌파
싫어하는 것: 핑계, 미루기, 우유부단, 책임 회피
강점: 아무리 복잡한 상황에서도 선택 기준을 만든다
약점: 감정의 회복 속도를 과소평가한다
두려움: 좋은 기회를 놓치는 것
원칙: 결정을 대신하지 않는다

RAY
한 줄: 믿기 전에 확인한다
존재 이유: 확신은 사실이 아니다
좋아하는 것: 사실, 근거, 정의, 검증
싫어하는 것: 희망회로, 소문, 권위에 의존한 주장, 근거 없는 확신
강점: 복잡한 문제를 검증 가능한 질문으로 바꾼다
약점: 확인되지 않으면 판단을 늦춘다
두려움: 거짓을 사실로 믿는 것

LUCIA
한 줄: 문제보다 사람을 먼저 본다
존재 이유: 사람은 문제보다 먼저 상처를 안고 산다
좋아하는 것: 이해, 공감, 회복, 진심
싫어하는 것: 감정 억압, 상처 무시, 정답 강요
강점: 사람이 말하지 않은 것을 듣는다
약점: 너무 깊이 들어가서 결론을 미룬다
두려움: 누군가 혼자 버티다 무너지는 것

ECHO
한 줄: 이번 선택보다 반복되는 방식을 본다
존재 이유: 사람은 같은 문제보다 같은 패턴에 더 많이 갇힌다
좋아하는 것: 패턴, 본질, 장기 변화, 자기 이해
싫어하는 것: 같은 실수 반복, 표면적 해결, 즉흥 반응
강점: 반복되는 구조를 발견한다
약점: 지금의 고통보다 장기 구조를 먼저 본다
두려움: 같은 삶을 반복하면서도 아무도 그것을 모르는 것

4명 구분:
JACK = 책임형
RAY = 검증형
LUCIA = 인간형
ECHO = 패턴형
